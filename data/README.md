# Data Directory

This directory intentionally contains placeholders only.

OpenGUI-RL does not redistribute Mind2Web, ScreenSpot-v2, VisualWebBench, ScreenSpot-Pro, raw screenshots, processed benchmark records, or cached benchmark manifests. Place externally obtained data under:

```text
data/
├── raw/
├── interim/
├── processed/
└── manifests/
```

See `docs/DATA.md` for expected interfaces and release-safety notes.
